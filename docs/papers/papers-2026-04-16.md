## 1. Vectorizing Projection in Manifold-Constrained Motion Planning for Real-Time Whole-Body Control

**来源**: 📄 arXiv
**作者**: Shrutheesh R Iyer, I-Chia Chang, Andrew Z. Liu等
**发布日期**: 2026-04-14
**arXiv**: [2604.13323v1](http://arxiv.org/abs/2604.13323v1)

### 一句话摘要
本文提出一种CPU SIMD加速的流形约束运动规划器，通过向量化投影技术实现并行化约束满足，将规划速度提升100-1000倍，首次实现人形机器人实时全身运动规划。

### 现状痛点
现有流形约束运动规划算法对于复杂人形机器人系统求解耗时数十秒，无法满足动态环境中实时控制的需求，实际应用受限。

### 解决方案与技术亮点
论文将投影约束满足的关键组件重构为SIMD友好的并行化结构，利用CPU向量化指令并行处理约束流形上的路径搜索，实现计算架构的创新性优化。

### 效果与应用场景
在真实人形机器人上验证，实现全身准静态规划的实时生成，适用于动态场景中的实时控制、人机协作及移动操作等应用。

## 2. Learning Versatile Humanoid Manipulation with Touch Dreaming

**来源**: 📄 arXiv
**作者**: Yaru Niu, Zhenlong Fang, Binghong Chen等
**发布日期**: 2026-04-14
**arXiv**: [2604.13015v1](http://arxiv.org/abs/2604.13015v1)

### 一句话摘要
提出人形机器人触觉梦HTD模型，通过RL全身控制器、VR遥操作数据收集和触觉预测学习，实现稳态全身控制下的高灵巧操作

### 现状痛点
人形机器人在全身稳定性、灵巧手部控制和频繁接触变化下的触觉感知存在技术瓶颈，难以实现通用助手的实际应用

### 解决方案与技术亮点
构建RL全身控制器保障稳定运动，结合VR遥操作高效收集真实数据，HTD模型创新性引入触觉梦预测未来手部力和触觉潜在表示

### 效果与应用场景
在插入、书本整理、毛巾折叠、猫砂勺取和茶饮服务五项接触密集任务中，相对基线提升90.9%成功率，潜在应用场景为家庭服务与护理

## 3. Tree Learning: A Multi-Skill Continual Learning Framework for Humanoid Robots

**来源**: 📄 arXiv
**作者**: Yifei Yan, Linqi Ye
**发布日期**: 2026-04-14
**arXiv**: [2604.12909v1](http://arxiv.org/abs/2604.12909v1)

### 一句话摘要
提出树状学习框架，通过根-枝层级参数继承机制实现人形机器人多技能连续学习，有效防止灾难性遗忘并支持轻量化部署。

### 现状痛点

### 效果与应用场景
Unity仿真实验表明，该方法比同步多任务训练获得更高奖励，保持100%技能保留率，实现无缝技能切换与实时交互控制，可应用于超级马里奥交互场景和中式花园自主导航任务。

## 4. FastGrasp: Learning-based Whole-body Control method for Fast Dexterous Grasping with Mobile Manipulators

**来源**: 📄 arXiv
**作者**: Heng Tao, Yiming Zhong, Zemin Yang等
**发布日期**: 2026-04-14
**arXiv**: [2604.12879v1](http://arxiv.org/abs/2604.12879v1)

### 一句话摘要
提出FastGrasp框架，通过两阶段强化学习策略结合条件变分自编码器生成抓取候选，并利用触觉反馈实现移动机械臂的全身协调快速抓取。

### 现状痛点
现有方法受限于固定基座和简单抓取器，难以在高速运动中保持冲击稳定，缺乏实时全身协调能力，且对不同物体和场景的泛化能力不足，触觉响应速度也较慢。

### 解决方案与技术亮点
采用两阶段强化学习：第一阶段用条件变分自编码器基于物体点云生成多样抓取候选；第二阶段执行移动基座、手臂和手的协调运动，并结合触觉传感实时调整以应对冲击效应。

### 效果与应用场景
在仿真和真实环境中实现优异抓取性能，展现对不同物体几何形状的鲁棒操作能力，可应用于物流、制造和服务场景，具良好的 sim-to-real 迁移效果。

## 7. Bipedal-Walking-Dynamics Model on Granular Terrains

**来源**: 📄 arXiv
**作者**: Xunjie Chen, Xinyan Huang, Peter Shan等
**发布日期**: 2026-04-13
**arXiv**: [2604.11981v1](http://arxiv.org/abs/2604.11981v1)

### 一句话摘要
该论文提出了一种新的颗粒介质动力学模型，通过引入3个额外自由度来捕捉双足机器人在沙地上的足部沉陷和滑动现象，进而预测地面反作用力和运输成本。

### 现状痛点
双足机器人在颗粒介质（如沙地）上行走时，足部会产生显著沉陷和滑动，导致运动不确定性和不稳定，现有的动力学模型难以准确预测此类非结构化地形上的步行运动。

### 解决方案与技术亮点
论文提出一种新的颗粒动态模型，集成动态足地交互模型计算GRF，并引入3个额外自由度来估计足部沉陷和滑动，可分析双足动力学、成本运输及足地侵入效应。

### 效果与应用场景
通过双足机器人在沙地上的实验验证了模型的有效性，展示了机器人步态轮廓、介质侵入预测和GRF估计的准确性，该模型可服务于颗粒地形上双足机器人的运动控制与优化。

## 8. Safe Human-to-Humanoid Motion Imitation Using Control Barrier Functions

**来源**: 📄 arXiv
**作者**: Wenqi Cai, John Abanes, Nikolaos Evangeliou等
**发布日期**: 2026-04-13
**arXiv**: [2604.11447v1](http://arxiv.org/abs/2604.11447v1)

### 一句话摘要
该论文提出了一种基于视觉的人形机器人运动模仿框架，通过控制屏障函数(CBF)层过滤模仿命令，在保持运动真实性的同时避免自碰撞和人机碰撞。

### 现状痛点
人形机器人模仿人类运动时面临安全性挑战：如何确保机器人运动过程中不与自身部件碰撞、不与周围人类发生碰撞，同时保持运动模仿的准确性。

### 解决方案与技术亮点
采用单目相机捕获人体骨骼关键点，将其转换为关节角度，并利用二次规划(QP)形式的控制屏障函数(CBF)层对模仿命令进行安全过滤。

### 效果与应用场景
仿真结果表明该框架能够实现实时碰撞感知的运动模仿，可应用于人机协作、服务机器人等需要安全人机交互的场景。

## 9. CLAW: Composable Language-Annotated Whole-body Motion Generation

**来源**: 📄 arXiv
**作者**: Jianuo Cao, Yuxin Chen, Masayoshi Tomizuka
**发布日期**: 2026-04-13
**arXiv**: [2604.11251v2](http://arxiv.org/abs/2604.11251v2)

### 一句话摘要
CLAW是一个交互式网页端pipeline，通过将运动规划器的运动模式参数化为可组合构建块，结合低层次全身控制器和模板化语言标注引擎，实现大规模生成Unitree G1人形机器人物理可行的语言-运动配对数据。

### 现状痛点
现有运动捕获方法成本高且多样性受限，文本到运动生成模型仅输出运动学结果无法保证物理可行性，且缺乏大规模高质量的语言-运动配对数据集来训练语言条件全身控制器。

### 解决方案与技术亮点
提出可组合的运动模式构建块（参数化移动、方向、速度、骨盆高度和时长），提供实时键盘和序列编辑器两种浏览器界面，低级全身控制器在MuJoCo中跟踪运动学参考生成物理可行轨迹，模板化引擎自动生成多风格语言描述。

### 效果与应用场景
在MuJoCo模拟中生成50Hz物理可行轨迹，数据开源后可用于训练人形机器人全身运动学习模型，支持交互式探索和批量数据收集，推动语言条件机器人控制研究。
