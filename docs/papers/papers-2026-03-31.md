## 1. Active Stereo-Camera Outperforms Multi-Sensor Setup in ACT Imitation Learning for Humanoid Manipulation

**来源**: 📄 arXiv
**作者**: Robin Kühn, Moritz Schappler, Thomas Seel等
**发布日期**: 2026-03-30
**arXiv**: [2603.28422v1](http://arxiv.org/abs/2603.28422v1)

### 一句话摘要
该论文在Unitree G1人形机器人上系统评估了14种传感器组合，发现 minimalist 的主动双目相机配置在数据受限场景下优于复杂多传感器方案，在空间泛化任务达87.5%成功率，结构化操作任务达94.4%，验证了传感器策略性选择对模仿学习效率的关键作用。

### 现状痛点
当前人形机器人教学缺乏传感器配置共识，多传感器增加复杂度与计算开销但未必提升性能，在有限数据下额外模态反而因信噪比低而降低成功率，行业尚未明确操作任务的最优感官硬件配置标准。

### 解决方案与技术亮点
提出开源 Unified Ablation Framework，通过传感器掩码在统一 master dataset 上系统消融评估14种组合，创新性地验证主动视觉在数据受限 regime 的优越性，发现 minimal 主动双目相机超越复杂配置，揭示触觉传感器需更大数据集才能生效的规律。

### 效果与应用场景
实验证明主动双目相机在空间泛化（87.5%）和结构化操作（94.4%）任务中的卓越性能，适用于数据稀缺的机器人教学场景，为高效人形机器人硬件设计提供指导，降低计算开销并提升学习鲁棒性。

## 2. Cost-Matching Model Predictive Control for Efficient Reinforcement Learning in Humanoid Locomotion

**来源**: 📄 arXiv
**作者**: Wenqi Cai, Kyriakos G. Vamvoudakis, Sébastien Gros等
**发布日期**: 2026-03-30
**arXiv**: [2603.28243v1](http://arxiv.org/abs/2603.28243v1)

### 一句话摘要
该论文提出一种成本匹配方法，通过训练参数化MPC使其近似强化学习中的动作值函数，从而实现高效的人形机器人运动控制梯度学习，避免了训练过程中重复求解MPC的计算负担。

### 现状痛点
人形机器人运动控制中，手动调整MPC成本参数耗时且困难；传统MPC-based RL需要在训练中反复求解MPC导致计算开销巨大；手动调优的基线方法在模型不匹配和外部扰动下鲁棒性不足。

### 解决方案与技术亮点
提出成本匹配框架，利用参数化MPC结合质心动力学，在记录的state-action轨迹上评估MPC成本-to-go，通过梯度下降最小化MPC预测值与实际回报之间的差异，从而高效学习最优成本参数。

### 效果与应用场景
在商业人形机器人平台仿真验证，显示相比手动调优基线具有更优的运动性能和更强的鲁棒性，可应用于双足机器人、康复辅助机器人及复杂地形自主导航等领域。
