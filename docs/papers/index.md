# 学术论文

这里汇集了人形机器人领域的最新学术研究论文，每篇论文都经过 Claude AI 的深度分析。

## 最新论文

论文将在数据收集脚本运行后自动出现在这里。

你可以运行以下命令手动收集论文:

```bash
python scripts/collect_papers.py
```

## 分析维度

每篇论文包含以下分析维度:

- **一句话摘要**: 论文核心贡献
- **现状痛点**: 要解决的问题或技术瓶颈
- **解决方案与技术亮点**: 创新方法和技术要点
- **效果与应用场景**: 实验效果和潜在应用
