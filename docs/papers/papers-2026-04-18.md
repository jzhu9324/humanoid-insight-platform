## 1. Switch: Learning Agile Skills Switching for Humanoid Robots

**来源**: 📄 arXiv
**作者**: Yuen-Fui Lau, Qihan Zhao, Yinhuai Wang等
**发布日期**: 2026-04-16
**arXiv**: [2604.14834v1](http://arxiv.org/abs/2604.14834v1)

### 一句话摘要
提出Switch分层多技能系统，通过技能图建立跨技能转换可能性，结合深度强化学习训练全身跟踪策略与在线调度器，实现人形机器人任意时刻的敏捷技能切换。

### 现状痛点
现有全身控制方法虽能实现多种运动技能，但缺乏灵活的技能间过渡能力，导致机器人切换技能时稳定性差、存在安全隐患，限制了实际应用。

### 解决方案与技术亮点
构建技能图(SG)基于运动学相似性建模跨技能转换关系；使用深度强化学习在技能图上训练全身跟踪策略；开发在线技能调度器执行图搜索以实时规划最优转换路径。

### 效果与应用场景
实验表明Switch能以高成功率执行敏捷技能转换，同时保持较强的运动模仿性能，适用于复杂环境下的双足运动控制与实时技能切换场景。

## 2. Vectorizing Projection in Manifold-Constrained Motion Planning for Real-Time Whole-Body Control

**来源**: 📄 arXiv
**作者**: Shrutheesh R Iyer, I-Chia Chang, Andrew Z. Liu等
**发布日期**: 2026-04-14
**arXiv**: [2604.13323v1](http://arxiv.org/abs/2604.13323v1)

### 一句话摘要
本文提出一种CPU SIMD加速的流形约束运动规划器，通过将约束满足问题转化为并行化结构，实现100-1000倍加速，使人形机器人实时全身控制成为可能。

### 现状痛点
现有流形约束规划算法在复杂系统（如人形机器人）上求解时间长达数十秒，难以满足动态环境中实时控制的需求，计算瓶颈严重制约了实际应用。

### 解决方案与技术亮点
提出向量化投影方法，将约束投影相关组件转化为SIMD并行可执行结构，利用CPU单指令多数据并行化技术加速约束满足求解，突破传统串行计算瓶颈。

### 效果与应用场景
在真实人形机器人上实现实时全身准静态规划生成，展示了动态环境下的实时规划能力，可应用于需要快速约束满足的服务机器人、工业协作等场景。

## 3. Learning Versatile Humanoid Manipulation with Touch Dreaming

**来源**: 📄 arXiv
**作者**: Yaru Niu, Zhenlong Fang, Binghong Chen等
**发布日期**: 2026-04-14
**arXiv**: [2604.13015v1](http://arxiv.org/abs/2604.13015v1)

### 一句话摘要
提出Touch Dreaming框架，通过预测未来手部关节力和触觉潜在表示，让Transformer学习接触感知表征，实现人形机器人灵巧操作，在5个接触-rich任务中达到平均90.9%的相对提升。

### 现状痛点
人形机器人在全身稳定控制、频繁接触变化下的感知与操作方面存在技术瓶颈，现有方法缺乏触觉与视觉、本体感觉的多模态融合感知能力，导致操作精度和成功率不足。

### 解决方案与技术亮点
开发RL全身控制器保证稳定运动，结合VR遥操作收集真实数据，提出HTD多模态Transformer创新性地整合触觉、视觉和本体感觉，并通过触觉 dreaming预测未来接触信息增强接触感知表征学习。

### 效果与应用场景
在Insert-T、图书整理、毛巾折叠、猫砂清理、茶水服务等5个接触密集任务中验证，实现90.9%相对基线的成功率提升，30%相对增益来自潜在空间触觉预测，展示了高效能的全身灵巧操作性能。

## 4. Tree Learning: A Multi-Skill Continual Learning Framework for Humanoid Robots

**来源**: 📄 arXiv
**作者**: Yifei Yan, Linqi Ye
**发布日期**: 2026-04-14
**arXiv**: [2604.12909v1](http://arxiv.org/abs/2604.12909v1)

### 一句话摘要
本文提出Tree Learning框架，通过根-分支层次参数继承机制实现人形机器人的多技能持续学习，在避免灾难性遗忘的同时高效扩展新技能，在Unity模拟中实现100%技能保留率并提升多任务奖励。

### 现状痛点
现有方法在多技能学习中存在灾难性遗忘问题，MoE模型需要复杂拓扑调整，大规模模型训练成本高，难以实现轻量级部署和实时技能切换。

### 解决方案与技术亮点
提出根-分支层次参数继承机制提供运动先验，多模态前馈适应机制结合相位调制和插值支持周期/非周期运动，任务级奖励塑造策略加速技能收敛，实现参数复用和技能扩展。

### 效果与应用场景
在Unity模拟中实现100%技能保留率，显著提升多种运动技能奖励，支持无缝多技能切换和实时交互控制，验证于超级马里奥互动场景和中国古典园林自主导航任务。

## 5. FastGrasp: Learning-based Whole-body Control method for Fast Dexterous Grasping with Mobile Manipulators

**来源**: 📄 arXiv
**作者**: Heng Tao, Yiming Zhong, Zemin Yang等
**发布日期**: 2026-04-14
**arXiv**: [2604.12879v1](http://arxiv.org/abs/2604.12879v1)

### 一句话摘要
本文提出FastGrasp框架，通过两阶段强化学习和条件变分自编码器实现移动机械臂的快速灵巧抓取，并结合触觉反馈实现实时抓取调整。

### 现状痛点
现有方法在高速运动下的冲击稳定化、实时全身协调、不同物体和场景的泛化能力方面存在瓶颈，且受限于固定基座、简单夹爪和慢速触觉响应。

### 解决方案与技术亮点
提出两阶段强化学习策略：第一阶段用条件变分自编码器基于物体点云生成多样化抓取候选；第二阶段执行移动基座、手臂和手的协调运动，并利用触觉传感进行实时抓取调整。

### 效果与应用场景
在模拟和真实环境中实现优越抓取性能，对不同几何形状物体具有稳健操作能力，具有良好的模拟到真实迁移效果，适用于物流、制造和服务场景。

## 6. Whole-Body Mobile Manipulation using Offline Reinforcement Learning on Sub-optimal Controllers

**来源**: 📄 arXiv
**作者**: Snehal Jauhri, Vignesh Prasad, Georgia Chalvatzaki
**发布日期**: 2026-04-14
**arXiv**: [2604.12509v1](http://arxiv.org/abs/2604.12509v1)

### 一句话摘要
论文提出WHOLE-MoMa框架，利用次优全身控制器作为结构先验生成演示数据，再通过离线强化学习和Q-chunking技术改进策略，实现无需真实世界数据的移动操作任务。

### 现状痛点
传统全身控制器需要大量手调参数且泛化性差，学习方法依赖昂贵的遥操作数据或复杂奖励工程，如何在不收集真实机器人数据的情况下提升次优控制器的性能是关键挑战。

### 解决方案与技术亮点
提出两阶段pipeline：第一阶段随机化轻量级WBC生成多样化演示数据；第二阶段结合离线RL和Q-chunking进行分块级critic评估与优势加权策略提取，实现行为改进与迁移。

### 效果与应用场景
在TIAGo++机器人模拟实验中，策略直接迁移到真实机器人，双手抽屉操作达80%成功率，橱柜+放置任务达68%成功率，无需任何真实世界训练数据。

## 7. Asymptotically Stable Gait Generation and Instantaneous Walkability Determination for Planar Almost Linear Biped with Knees

**来源**: 📄 arXiv
**作者**: Fumihiko Asano, Ning Lei, Taiki Sedoguchi
**发布日期**: 2026-04-14
**arXiv**: [2604.12274v1](http://arxiv.org/abs/2604.12274v1)

### 一句话摘要
论文提出一种针对平面带膝关节双足机器人的渐近稳定步态生成方法，通过将重力项在髋关节角度附近线性近似，实现无需数值积分的即时步态计算和步行能力判定。

### 现状痛点
传统双足机器人步态计算需要耗时的数值积分，难以实现即时步行能力判定，且在下台阶等复杂地形时存在计算精度和响应速度的瓶颈。

### 解决方案与技术亮点
论文推导出6-DOF双足机器人运动方程和碰撞模型，简化至3-DOF模型，将重力项进行泰勒展开线性近似，通过状态空间实现和离散化实现迭代计算，突破传统数值积分的耗时限制。

### 效果与应用场景
数值仿真验证了步态的渐近稳定性和即时计算能力（计算时间可忽略），膝关节弯曲角度和展开点选择影响近似精度。该方法适用于双足机器人实时步态规划和步行能力快速评估。
